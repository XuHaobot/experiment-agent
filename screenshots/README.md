# Screenshots

This directory is reserved for project screenshots used in the main README.

Do not add generated placeholder or fake screenshots. Add real screenshots from the running Streamlit app when they are available.

Recommended files:

- `upload.png`: 上传实验记录页面
- `agent_trace.png`: Agent 工具调用过程
- `structured_json.png`: 结构化 JSON 输出
- `report.png`: Markdown 实验复盘报告
- `graph_entities.png`: 知识图谱实体关系列表
- `graph_visualization.png`: pyvis 图谱可视化
